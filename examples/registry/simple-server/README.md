# simple-server

## Getting started

### Installation

Install dependencies for the module before use:
```sh
# with npm
$ npm install

# or yarn
$ yarn install
```

### Environment

You can clone example environment by comamnd:
```sh
$ cp .env.example .env
```

### Scripts

You can get started quickly with the following script.

#### Start development mode
```sh
# with npm
$ npm run dev

# or yarn
$ yarn dev
```

#### Start production mode
```sh
# with npm
$ npm run start

# or yarn
$ yarn start
```
