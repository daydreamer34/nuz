# registry-server

## Getting started

### Installation

```sh
# with npm
$ npm install

# or yarn
$ yarn install
```

### Start development mode

```sh
# with npm
$ npm run dev

# or yarn
$ yarn dev
```

### Start production mode

```sh
# with npm
$ npm run start

# or yarn
$ yarn start
```
